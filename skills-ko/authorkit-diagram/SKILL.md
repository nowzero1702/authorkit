# authorkit.diagram — 도해 제작

텍스트 블록도를 md 코드 블록 형태로 생성합니다.

## 4가지 모드

### 1. 레퍼런스 기반
레퍼런스의 특정 그림을 텍스트 블록도로 재구성합니다.
- "fig-4.34를 텍스트 블록도로 만들어줘"

### 2. 글 내용 기반
본문 텍스트에서 도해 가능한 패턴을 탐색하여 생성합니다.
탐색 패턴: 순서/단계, 비교, 구성요소, 데이터흐름, 계층/분류, 타이밍

### 3. [그림] 참조 자동 생성
원고에 `[그림]` 텍스트만 있고 도해가 없는 위치에 자동 생성합니다.

### 4. scan 모드
특정 절을 스캔하여 도해로 만들면 좋을 부분을 제안합니다.
- "6-2절에서 도해 가능한 부분 찾아줘"

## 출력 형식

모든 도해는 md 코드 블록 안의 텍스트 블록도로 출력됩니다.

기본 요소:
- 박스: `┌──┐ │  │ └──┘`
- 화살표: `──▶` (데이터), `══▶` (강조), `─ ─▶` (비활성)
- 구분선: `║  ║` (파이프라인 레지스터)
- 번호: `① ② ③`

복잡도는 constitution의 대상 독자 수준에 따라 자동 조절합니다.
