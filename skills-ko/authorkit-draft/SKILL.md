# authorkit.draft — 절 단위 집필/퇴고

원고의 특정 절을 old/new 비교 형태로 퇴고하거나 새로 작성합니다.

## 사용법 예시

- "5-1절 퇴고해줘" → 자동 수준 판단
- "5-1 서식만 정리해줘" → 수준 1
- "6-1 새로 써줘" → 수준 4 (전면 재작성)

## 퇴고 4단계

| 수준 | 이름 | 내용 |
|:----:|------|------|
| 1 | 서식 통일 | 띄어쓰기, 용어, 표기만 정리 |
| 2 | 내용 보강 | 텍스트 블록도 추가, 설명 보충, 이지스팁 |
| 3 | 구조 변경 | 소절 추가/삭제, 순서 변경 |
| 4 | 전면 재작성 | 기존 비유/예제 살리되 전체를 새로 쓰기 |

사용자가 수준을 지정하지 않으면 원고 상태를 분석하여 자동 판단합니다.

## 실행 흐름

1. 기존 원고에서 해당 절 추출 → `drafts/chXX/절번호-old.md`
2. constitution + glossary 기준으로 퇴고/작성 → `drafts/chXX/절번호-new.md`
3. compare 결과가 있으면 레퍼런스 내용 반영
4. 텍스트 블록도 삽입 (필요 시)
5. 변경 사항 요약 표 출력

## 버전 관리

다시 draft를 실행하면:
- 기존 new → old로 이동
- 새 수정본 → new로 생성
- 이전 old → past/ 폴더로 아카이브

## 문체 적용

constitution.md의 규칙을 자동 적용합니다:
- 종결 어미, 존대 수준, 용어 표기, 보충 요소 문체
