# authorkit.review — 문체/용어/구조 검증

constitution과 glossary를 기준으로 원고를 검증합니다.

## 4가지 검증 항목

### 1. 문체 (style)
- 종결 어미, 존대 수준, 보충 요소 문체, 느낌표/이모지
- constitution.md 기준과 대조

### 2. 용어 (glossary)
- 첫 등장 표기, 이후 표기, 동일 개념 다중 번역, 미등록 용어
- glossary.md 기준과 대조

### 3. 상호참조 (xref)
- 챕터/절 참조 번호 정확성
- 전방참조, 후방참조, 그림 참조, 레퍼런스 참조

### 4. 개연성 (coherence)
- 선수지식 순서, 전방참조 모순, 예고 이행
- 깊이 일관성, 비유 일관성, 도입/마무리 존재

## 사용법 예시

- "전체 검증해줘" → 4항목 모두
- "6-2절 문체 확인해줘" → 특정 절 + 문체만
- "용어 일관성 확인해줘" → 전체 + 용어만

## 산출물

심각도 3단계(높음/중간/낮음) 종합 리포트를 출력합니다.
자동 수정 가능한 항목은 `/authorkit.draft [절번호] 서식`으로 수정할 수 있도록 안내합니다.

리포트는 `authorkit/reviews/review-[DATE].md`에 저장됩니다.
